# Student Record Management System (C)

This is a simple Student Record Management System built using C and Linked List.

## Features

* Add Student
* Display Students
* Search Student
* Update Student
* Delete Student

## Concepts Used

* Structures
* Pointers
* Linked List
* Dynamic Memory Allocation

## How to Run

1. Compile the code:

```
gcc main.c -o student
```

2. Run:

```
./student
```
